# frc-software-2015

This is FRC team 3061’s readme for the 2015 season. Code was written in labview, with different branches containing code for different areas. Additional information for each branch can be found in the comments in the .vi files in the respective branch.


Branches

`master` contains the code that was actually run on our robot at the 2015 Little Rock and Midwest Regionals.

all other branches are stale/no longer in use


Directory Structure

`frc-software-2015/Dashboard` contains code to run on the Dashboard

`frc-software-2015/Icons` contains .png images of custom vi icons

`frc-software-2015/Robot` contains vi’s to control all aspects of the robot, split into directory by robot mode and funcionality

`frc-software-2016/Simulation` contains .ive and .stl files used to simulate aspects of the robot. More information can be found in the readme file located inside of it. 

`frc-software-2015/RecycleRush/Builds` contains a link to the NI bin

`frc-software-2015/builds/FRC_Dashboard` contains final changes to dashboard


Files

`Alex Fails.vi` contains experimental autonomous and teleop drives

`Clapper Setpoints.txt` contains the setpoints for the clapper, in readable format


More Information can be found in the comments and documentation in each folder
